# Steps to Run

1) Clone the git repository. Use this command:

> git clone https://github.com/AnshuSharma111/ExpenseTracker.git
> cd ExpenseTracker

2) Install Dependencies

> cd backend
> npm install

Back out of the backend folder and cd to the frontend folder and start it.

> cd ../frontend
> npm install

3) Start the frontend application. Still in the frontend folder, run the command.

> npm start

4) Open another terminal and start the backend application. Navigate to the backend folder and start it

> cd ../backend 
> npm run test

5) The servers should now be running on:
frontend: http://localhost:3000
backend:  http://localhost:8000

And you are ready to go!
